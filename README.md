# fantastic-music-player

#### 介绍
Music player with a fancy ui, powered by bass.dll

#### 软件架构

使用GDI绘制界面，使用Bass.dll作为引擎

#### 安装教程

1.  打开发行版页面
2.  下载最新的exe
3.  放到一个有音乐的文件夹双击打开

#### 使用说明

1.  建议慢慢摸索
2.  每个按钮都点点看
3.  使用很简单的
